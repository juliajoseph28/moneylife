## ğŸ“– About

MoneyLife is an engaging web-based game that simulates real-world financial decisions and their consequences. Players navigate through various life scenarios, making choices about spending, saving, investing, and managing their finances. The game provides a safe, interactive environment to develop financial literacy skills without real-world risks.

### Key Features

- **Interactive Gameplay** - Make financial decisions and see their immediate and long-term impacts
- **Real-World Scenarios** - Face authentic financial situations including budgeting, investing, debt management, and emergency planning
- **Educational Focus** - Learn practical money management skills through engaging gameplay
- **Progress Tracking** - Monitor your financial health and decision-making patterns over time
- **Responsive Design** - Play seamlessly across desktop, tablet, and mobile devices

## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/moneylife.git
cd moneylife
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm run dev
# or
yarn dev
```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal)

### Build for Production

```bash
npm run build
# or
yarn build
```
